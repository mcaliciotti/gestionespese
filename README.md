# Libro Mastro

App PWA per gestione conti, carte e scadenze personali. Tutti i dati restano in locale sul dispositivo.

## Come pubblicarla su GitHub Pages

1. Crea un repository pubblico su GitHub (qualsiasi nome, es. `mastro`)
2. Carica il file `index.html` alla **radice** del repo (NON dentro cartelle)
3. Settings → Pages → Source: `Deploy from a branch` → Branch: `main` / `(root)` → Save
4. Aspetta 1-2 minuti, l'URL sarà: `https://TUOUSER.github.io/NOMEREPO/`

## Come installarla su Android

1. Apri l'URL su Chrome o Brave
2. Menu (⋮) → "Aggiungi a schermata Home" / "Installa app"
3. Funziona offline, dati persistenti nel browser

## Backup

Setup → "Esporta tutto in JSON" — fai backup regolari, i dati stanno solo nel browser.
