# Disponi

App PWA per gestione conti, carte e scadenze personali. Ti dice sempre di quanto **disponi davvero**, al netto delle scadenze future. Tutti i dati restano in locale sul dispositivo.

## Struttura file

- `index.html` — l'app
- `manifest.json` — configurazione PWA
- `icon-192.png` / `icon-512.png` — icone dell'app
- `icon-maskable-512.png` — icona per Android adaptive icons
- `apple-touch-icon.png` — icona per iOS / alcuni browser Android
- `favicon-32.png` — icona tab browser
- `icon.svg` — sorgente vettoriale (opzionale)

## Pubblicazione su GitHub Pages

1. Crea repository pubblico
2. Carica **tutti** i file alla **radice** del repo
3. Settings → Pages → Branch `main` / `(root)` → Save
4. URL: `https://TUOUSER.github.io/NOMEREPO/`

## Installazione Android

1. Apri l'URL su Chrome/Brave
2. Menu ⋮ → "Aggiungi a schermata Home"
3. Funziona offline, dati nel browser

## Backup

Setup → "Esporta tutto in JSON" — fai backup regolari.
